## Die Plattform ist blockiert, die Website darf es nicht sein

Nuvora Studio macht eine einzige Sache: LinkedIn zum wichtigsten Vertriebskanal von B2B-Unternehmen machen. Mit einem offensichtlichen Haken: LinkedIn funktioniert in China kaum.

Das China-Problem verschwindet dadurch nicht. Nuvoras Team arbeitet teils vom Festland aus, und viele der Konzernentscheider, die es anspricht, ebenfalls. Die Website läuft in fünf Sprachen, Chinesisch inklusive. Das Produkt, das sie verkauft, lebt also auf einer Plattform, die die Firewall blockiert, doch die Website, die es verkauft, muss in Shanghai reibungslos öffnen.

Das tat sie nicht. Im Ausland gehostet, gestützt auf die üblichen Dienste von anderswo, lud sie vom Festland aus langsam und stolperte über einige Ressourcen, die die Firewall drosselt. Dazu kam ein kostenloses Tool zur Profiloptimierung, das funktionieren musste, egal wo der Besucher saß.

| | |
|---|---|
| **Kunde** | Nuvora Studio |
| **Branche** | B2B-Marketing-Agentur für LinkedIn |
| **Sprachen** | Englisch, Französisch, Chinesisch, Spanisch, Deutsch |
| **Plattform** | Website mit interaktivem Tool |
| **Kernleistungen** | Design-Umsetzung, Integration, Hosting in China, GEO, Wartung |

## Der Ansatz: schnell auf dem Festland, Tool inklusive

Wir bauten die Website auf Hosting in China neu, erledigten die ICP-Registrierung und entfernten die ausländischen Abhängigkeiten, die die Ladezeiten für Besucher vom Festland belasteten. Die fünf Sprachen blieben, verbunden mit sauberem hreflang-Markup, damit jede eigenständig funktioniert.

Das Tool zur Profiloptimierung brauchte Sorgfalt. Ein interaktives Tool ist genau das, was hinter der Firewall stillschweigend bricht, wenn es einen blockierten Dienst aufruft. Also bauten wir es neu, damit es auf einer aus China erreichbaren Infrastruktur läuft.

Außerdem richteten wir die gesamte Website auf die KI-Suche aus. Nuvoras Käufer wenden sich zunehmend an einen Assistenten, der sich mit B2B auf LinkedIn auskennt, bevor sie einen Kollegen fragen, und wir wollten, dass die Antwort Nuvora lautet. Strukturiert, zitierfähig, maschinenlesbar.

## Das Ergebnis: eine Website, die dort öffnet, wo ihre Käufer sind

Nuvoras Website lädt jetzt schnell aus China, Tool inklusive, vollständig registriert, ohne dass noch etwas auf einen blockierten Server wartet. Das Team auf dem Festland und die Interessenten in China bekommen dieselbe schnelle Website wie alle anderen.

Es liegt eine kleine Ironie darin, für eine LinkedIn-Agentur eine schnelle China-Website zu bauen. Sie zählt trotzdem, denn wer über die Beauftragung von Nuvora entscheidet, sitzt oft genau hier, in China.

> "Die Hälfte unserer Käufer ist in China, auch wenn LinkedIn es nicht ist. Die Website musste für sie laden. Das tut sie jetzt, und schneller als die Version davor."
>
> **Marcus T.**, Mitgründer, Nuvora Studio
