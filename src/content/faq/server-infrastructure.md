Alibaba Cloud or Tencent Cloud, both with ICP-compliant mainland data centres. Stack: Linux (Ubuntu or CentOS), Nginx, PHP 8.x with OPcache, MySQL 8.0 or MariaDB, Redis for caching, and a China CDN layer. We pick the server region closest to your audience - Shanghai, Beijing, Shenzhen, among others.

Backups, server hardening, and uptime monitoring are configured before we migrate anything. That's part of the base setup, not an optional add-on.
