question: "What happens if the Great Firewall blocks our site after launch?"
order: 15
category: "technical"
---

Hosting on ICP-licensed servers inside mainland China makes a full block really unlikely. Content-level filtering can still get triggered though - that's a different mechanism. We run monitoring from several Chinese cities using local test tools and because we're here we catch things quickly. Agencies based outside China can spend days debugging through a VPN, which distorts everything. We're on the same network as your users, so we see exactly what they see.
