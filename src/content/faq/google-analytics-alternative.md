Baidu Tongji. It covers traffic sources, behaviour flow, conversion tracking, and event tracking. The bonus is it ties directly into the Baidu SEO ecosystem so search data and site analytics live in one dashboard.

For heatmaps and session recordings we install China-compatible tools alongside it. If GDPR is a concern we can deploy self-hosted Matomo on your Chinese server so data never touches a third party. Whatever the setup, analytics is live from day one.
